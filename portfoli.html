Aventura Travel

Solució integral per a agència de viatges

Per al client Aventura Travel, hem dissenyat i implementat una infraestructura tecnològica completa:

Desenvolupament Web

  • Lloc web creat amb WordPress, totalment personalitzat
  • Adaptat a mòbils i optimitzat per SEO
  • Integració de formularis de reserva i passarel·les de pagament

Infraestructura informàtica

  • Instal·lació i configuració de servidors Windows i Ubuntu
  • Creació de dominis, comptes d’usuari i control d’accés

Xarxa i hardware

  • Muntatge d’equips informàtics per a totes les oficines
  • Cablejat de xarxa estructurat per totes les estances
  • Verificació de la connectivitat i rendiment

Tecnologies emprades: WordPress, HTML, CSS, Windows Server, Ubuntu Server, UTP Cat6
